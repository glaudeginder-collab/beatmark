# BeatMark 🏆

> Is your portfolio beating the market? Find out in 60 seconds.

BeatMark lets you compare your investment portfolio against VWRL (Vanguard FTSE All-World UCITS ETF) — the gold standard low-cost global index ETF.

## The Question We Answer

Most people paying a wealth manager have no idea if they're actually beating the market. BeatMark shows you clearly: your portfolio return vs what you'd have if you'd simply bought VWRL.
